# Smart Complaint Routing System (Salesforce)

## 📌 Project Overview

This project is a Salesforce-based complaint management system that automatically routes customer complaints to the appropriate department using automation.

## 🚀 Features

* Custom Objects (Complaint, Customer)
* Lookup Relationship (Customer → Complaint)
* Record-Triggered Flow for automation
* Automatic Department Assignment
* Email Notifications
* Reports and Dashboards

## 🧠 Business Problem

Manual complaint handling leads to delays and misrouting. This system automates routing to improve response time and efficiency.

## ⚙️ Tech Stack

* Salesforce CRM
* Flow Builder
* Custom Objects & Fields
* Reports & Dashboards

## 🔄 Workflow

Customer creates complaint
→ Complaint linked to customer
→ Flow triggers
→ Department assigned automatically
→ Email notification sent
→ Data visualized in dashboard

## 📊 Dashboard

* Complaints by Type
* Complaints by Department
* Complaints by Priority
